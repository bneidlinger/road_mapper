# Scripts Directory

This directory contains utility scripts for development and deployment.

## Scripts

### deploy-gh-pages.sh
Deployment script for GitHub Pages. Builds the project and deploys to the `gh-pages` branch.

### Windows Batch Files
- `dev.bat` / `dev.cmd` - Start the development server on Windows
- `start.bat` - Quick start script for Windows
- `start-direct.bat` - Direct start without npm (Windows)

## Usage

### Deploy to GitHub Pages
```bash
./scripts/deploy-gh-pages.sh
```

### Start Development Server (Windows)
```batch
scripts\dev.bat
```
