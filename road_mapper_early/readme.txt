# Road Mapper

Road Mapper is a simplified road and transportation network drawing application. It reuses concepts from Facility Map Creator while focusing on road design.

## Folder Structure

```
road-mapper/
├── src/
│   ├── modules/
│   │   ├── canvas/      # Rendering and interaction
│   │   ├── elements/    # Road and intersection definitions
│   │   ├── export/      # Export utilities
│   │   ├── settings/    # Application state and user settings
│   │   └── ui/          # Basic UI components
│   ├── styles/          # CSS styles
│   └── app.js           # Application entry point
├── public/
│   └── index.html       # Demo page
└── README.md
```

This skeleton provides a starting point for building the dedicated road mapping tool described in the planning outline.
